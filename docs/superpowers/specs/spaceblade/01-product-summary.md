# Product Summary

`Spaceblade` is a one-key 2D browser action game for the Hack Club OneKey challenge. The player controls a cyber swordsman in a side-view neon arena and survives escalating waves using only the `Space` key for combat, menu navigation, and run flow.

The game should feel intentional, readable, and polished. Quality comes from tight combat timing, strong feedback, disciplined scope, and the one-key constraint feeling like the main mechanic.

Product goals:

- Deliver a complete playable browser game.
- Preserve the one-key constraint as the main design strength.
- Reach release quality through phased development and isolated tuning.
- Support an online leaderboard with low operating cost.
- Stay deployable as a static frontend with minimal backend usage.

Non-goals:

- No real-time multiplayer.
- No always-on gameplay backend.
- No complex account system in v1.
- No procedural campaign or branching world structure.
